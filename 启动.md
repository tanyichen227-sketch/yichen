如果你要把项目装到另一台电脑，按这套流程最稳：

安装基础环境
Python 3.10+、Node.js 18+、Git、Ollama

拷贝项目或克隆仓库
进入项目根目录（有 dev.ps1 的目录）

安装后端依赖

cd RagBackend
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
安装前端依赖
cd ..\RagFrontend
npm install
启动 Ollama 并准备模型
ollama serve
ollama pull qwen2:latest
你也可以拉 qwen2:0.5b，但现在即使没装，系统也会自动回退到可用模型。

一键启动项目（推荐）
cd ..
powershell -ExecutionPolicy Bypass -File .\dev.ps1
访问地址
前端：http://localhost:5173
后端：http://localhost:8000
接口文档：http://localhost:8000/docs

如果要迁移旧数据（可选）
把旧机器这些目录/文件复制到新机器同路径：

RagBackend/local-KLB-files
RagBackend/metadata
RagBackend/models_config.json
RagBackend/user_avatars
如果登录仍失败
先执行：

powershell -ExecutionPolicy Bypass -File .\dev.ps1 -Status
确认 MySQL 和后端都为正常状态。
