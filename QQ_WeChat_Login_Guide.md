## 一、QQ 登录（QQ 互联）

### 1.1 平台地址

- 管理中心：https://connect.qq.com
- 官方文档：https://wiki.connect.qq.com

### 1.2 是否支持个人开发者？

**✅ 支持。** QQ 互联同时支持**个人开发者**和**企业开发者**，个人用身份证实名认证即可申请。

### 1.3 申请条件

| 条件 | 说明 |
|---|---|
| QQ 账号 | 用于注册开发者身份，**建议使用长期稳定的 QQ 号**（一旦注册不可更换） |
| 实名认证 | 填写真实姓名 + 上传手持身份证照片（个人开发者） |
| 网站备案 | 国内网站需要 **ICP 备案号**；**海外服务器的网站无需备案**，可直接申请 |
| 回调域名 | 必须填写正式域名（不接受 `localhost`），需在 QQ 互联后台添加 |
| 应用图标 | 上传应用 Logo，规格 80×80px 以上 |

### 1.4 费用

| 项目 | 费用 |
|---|---|
| 注册开发者账号 | **免费** |
| 申请网站应用（获取 AppID / AppKey） | **免费** |
| 审核费用 | **免费** |
| QQ 互联接口调用 | **免费**（基础登录接口） |

> **结论：QQ 互联对个人开发者完全免费。**

### 1.5 审核时间

- 开发者实名认证审核：**1～24 小时**
- 网站应用审核：约 **5 个工作日**

> 审核通过前，AppID 只能供**开发者自己的 QQ 号**测试，无法让其他用户使用。

### 1.6 申请流程（网站应用）

```
第一步：注册开发者
  → 访问 https://connect.qq.com
  → 点击「成为开发者」→ 选择「个人」
  → 填写真实姓名 + 手持身份证照片
  → 等待审核（1～24 小时）

第二步：创建网站应用
  → 审核通过后，进入「管理中心」
  → 选择「网站应用」→「创建应用」
  → 填写：应用名称、应用简介、应用图标、回调域名、ICP 备案号（国内）
  → 提交审核

第三步：获取 AppID / AppKey
  → 审核通过后，在应用列表页查看
  → 将回调地址在后台更新为正式域名

第四步：配置回调地址
  → 格式示例：https://your-domain.com/api/qq/callback
  → localhost 不被接受，必须是正式域名
```

## 二、微信登录（微信开放平台）

### 2.1 平台地址

- 官网：https://open.weixin.qq.com
- 文档：https://developers.weixin.qq.com/doc/oplatform/

### 2.2 是否支持个人开发者？

**⚠️ 有条件支持，但限制较多。**

| 功能 | 个人开发者 | 企业/组织 |
|---|---|---|
| 微信小程序 | ✅ 支持（认证费 ¥30/年） | ✅ 支持（¥300/年） |
| 微信公众号 | ✅ 支持（订阅号，无认证费） | ✅ 支持 |
| **网站应用微信登录** | ❌ **个人开发者无法申请** | ✅ 需企业/组织资质 |
| 移动应用（App）微信登录 | ❌ 无法申请 | ✅ 支持 |

> **重要**：「网站应用微信登录」（即网页版的微信扫码登录）**只向企业/组织开发者开放，个人开发者无法申请**，这是微信官方的硬性规定，无技术手段绕过。

### 2.3 申请条件（企业/组织，网站应用）

| 条件 | 说明 |
|---|---|
| 主体资质 | 企业营业执照 / 政府机关 / 事业单位 / 民办非企业单位等 |
| 开发者资质认证 | 必须完成认证，需要法人扫脸（人脸识别） |
| 对公银行账户 | 用于认证身份（需打一笔小额验证款） |
| 已备案网站域名 | 必须是 **ICP 已备案的 HTTPS 域名** |
| AppID 申请后审核 | 提交资料后人工审核，约 **7 个工作日** |

### 2.4 费用

| 项目 | 金额 | 说明 |
|---|---|---|
| 开发者资质认证 | **¥300/年**（中国大陆） | 每年需续费，否则高级能力失效 |
| 网站应用申请 | **免费** | 认证通过后创建应用不额外收费 |
| 微信登录接口调用 | **免费** | 基础 OAuth2.0 接口 |
| 境外地区认证费 | **$99/年** | 适用于非中国大陆主体 |

> **结论：微信网站登录的核心门槛是 ¥300/年 的认证费 + 企业资质，个人无法参与。**

### 2.5 申请流程（仅供企业参考）

```
第一步：注册微信开放平台账号
  → 访问 https://open.weixin.qq.com
  → 使用企业邮箱注册

第二步：完成开发者资质认证
  → 进入「账号中心」→「开发者资质认证」
  → 选择主体类型（企业/政府/事业单位等）
  → 填写营业执照信息 + 法人人脸识别
  → 支付认证费 ¥300
  → 等待审核（约 7 个工作日）

第三步：创建网站应用
  → 审核通过后，进入「管理中心」→「网站应用」→「创建应用」
  → 填写：应用名称、应用官网（HTTPS）、ICP 备案号、应用描述
  → 提交审核

第四步：获取 AppID / AppSecret
  → 审核通过后可查看
  → 在系统后端配置 WECHAT_APP_ID 和 WECHAT_APP_SECRET

第五步：配置回调域名
  → 在应用详情中添加「已通过授权的域名」
  → 格式：your-domain.com（不含 https://）
```

### 2.6 个人开发者的替代方案

如果你是个人开发者，无法申请网站应用微信登录，可以考虑以下替代方案：

| 方案 | 可行性 | 说明 |
|---|---|---|
| 用手机号注册/登录 | ✅ 最简单 | 无需第三方授权，直接短信验证码登录 |
| 微信公众号网页授权 | ⚠️ 有条件 | 需要认证公众号（个人可申请订阅号，但网页授权需服务号），较复杂 |
| 接入其他第三方登录 | ✅ 推荐 | GitHub、Google、微博等平台对个人开发者更友好 |
| 等待项目转为企业运营 | 长期规划 | 获取营业执照后即可申请 |
